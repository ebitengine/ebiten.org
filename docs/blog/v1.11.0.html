v1.11.0 is released

v1.11.0 がリリースされました

Hajime Hoshi
2020-04-13

We are very happy to announce the release of v1.11.0 (Releaase Note). The new features are:

v1.11.0 がリリースされました! 詳しくはリリースノートを参照してください。新しい機能は次のとおりです:

  • Game interface and RunGame
  • Game インターフェイスと RunGame
  • Resizable window
  • サイズ可変ウィンドウ
  • Window position
  • ウィンドウ位置
  • Transparent screen
  • 透過スクリーン
  • Floating window
  • 常時最前面ウィンドウ
  • Capturing cursor
  • カーソルキャプチャ

You can develop a desktop mascot application with these API. See the mascot example.

これら API を使って、デスクトップマスコットアプリケーションが作れます。 mascot サンプルを参考にしてください。

Enjoy!
