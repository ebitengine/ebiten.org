Graphics (Basic)

Animation

Animation by changing the source region of an image by SubImage.

nn

Filter

Demonstrating the difference between nearest filter and linear filter.

Flood Fill

Filling images with solid colors by a composition mode.

Font

Rendering glyphs of a TrueType font by text.Draw.

HSV

Changing hue-saturation-value by specifying a color matrix.

Infinite Scroll

Infinite scroll by changing the rendering position frame by frame.

Game of Life

Plotting Conway's game of life by ReplacePixels.

Mandelbrot

Plotting a Mandelbrot set by ReplacePixels and complex numbers.

Mosaic

Mosaic effect by shrink an image once and enlarge it with nearest filter.

Noise

Rending noise by ReplacePixels.

Particles

Rendering bunch of images with geometry/color matrices.

Polygons

Rendering polygons by DrawTriangles.

Sprites

Rendering many images very fast by automatic batches.

Tiles

Rednering tiles with DrawImage specifying sub-images.

Hi-DPI

Masking

Masking by DrawImage and a composition mode.

Paint

Drawing tool using an offscreen image as a canvas.

Perspective

Pseudo raster scrolling by rendering thin parts of an image.

Ray Casting

Renderng ray casting regions.
